## Manual Verification

### Question

Average PM2.5 in Bihar

### Manual Calculation

The following Pandas computation was executed directly:

```python
df[
    (df["state"] == "Bihar")
    &
    (df["pollutant_id"] == "PM2.5")
]["pollutant_avg"].mean()
```
